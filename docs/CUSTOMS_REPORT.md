# Customs and Annual Brewing Reports

[Back to the Brauhaus overview](../README.md)

## Purpose

Brauhaus can generate annual PDF reports to support customs-related brewing documentation.

The report provides an overview of the relevant brews for a selected year.

## Report Content

Depending on the available brew data, the report can include:

- Brew date
- Beer or recipe name
- Brewed volume
- Original gravity
- Total brewed volume for the selected year

This makes it possible to document how much beer was brewed during the year and which original gravity values were associated with the individual brews.

## Typical Workflow

1. Record completed brews in Brauhaus.
2. Ensure that brewed volume and original gravity are stored.
3. Select the required reporting year.
4. Generate the annual PDF report.
5. Review the generated values before using the report.

## Data Quality

The report can only be as accurate as the stored brew data.

Before generating the PDF, verify:

- All relevant brews are included
- Brew dates are correct
- Brewed volumes are correct
- Original gravity values are present
- Duplicate brews are not included
- Test or cancelled brews are excluded when appropriate

## Legal Notice

The generated report supports the preparation of customs documentation.

It does not replace verification of the currently applicable legal, tax, or administrative requirements. The responsible person must review the report and ensure that the submitted information is complete and correct.

## Security

The report may contain operational production data.

Before sharing it, verify that it does not contain:

- Personal information
- Internal notes not intended for submission
- Confidential system information
- Internal network details
- Credentials or tokens
